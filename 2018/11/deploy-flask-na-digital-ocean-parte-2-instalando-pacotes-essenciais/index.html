Deploy Flask na Digital Ocean - Parte 2 - Instalando pacotes essênciais

Fri Nov 02, 2018 by Lucas Simon in
  Deploy Flask na Digital Ocean   devops   digital ocean , deploy , flask , python

Pacotes do servidor

Abaixo estão os pacotes essenciais para compilar os pacotes python, instalados via pip, como tamém pacotes para trabalharmos com o servidor.

# apt install build-essential wget vim zsh libffi-dev  libssl-dev libpcre3-dev zlib1g-dev libtiff5-dev libbz2-dev -y

Install git

Precisamos do git para baixar nosso repositório e um plugin chamado oh-my-zsh.

# apt install git -y

Libs para python Python 3

Aqui, instalamos os pacotes do python3, assim como os módulos pip e venv.

# apt install python3 python3-dev python3-pip python3-venv -y

Nginx

O nginx será nosso servidor web.

# apt install nginx -y

Supervisor

E o supervisor será para gerenciar nossa aplicação. Em outras artigo podemos abordar o systemd

# apt install supervisor -y
