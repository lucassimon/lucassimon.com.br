Sobre

Tue Jun 12, 2018 by Lucas Simon

Sou desenvolvedor web e amo arquiteturas de software e análise de sistemas. Utilizo Python como minha principal linguagem de programação em conjunto com o PostgresSQL / MySQL / MongoDB. Para melhorar a minha produtividade gosto de utilizar o Python com Django Framework ou Flask.

Atualmente, estou estudando arquiteturas de software com o GRAPHQL em conjunto com o NodeJS. Escolhi o NodeJs porque sinto a necessidade de melhorar minhas habilidades em JavaScript, dado que ele é usado no navegador, frameworks frontend como Vuejs, ReactJS e também para criar aplicativos desktop e celulares. Meu próximo objetivo é aprender sobre microsserviços e Queque Systems.
