Ola, vamos fazer uma breve introdução sobre o Elixir, uma linguagem de programação funcional. Elixir O Elixir é uma linguagem de programação funcional que foi desenvolvida pelo brasileiro José Valim. Essa linguagem tem como objetivo resolver problemas de escalabilidade, ser tolerante a falhas e alta concorrência.
