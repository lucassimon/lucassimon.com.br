About

Tue Jun 12, 2018 by Lucas Simon

I am a 30 years old Brazilian web developer who loves software architectures. I have Python as a main programming language. To create systems I use DBMS (PostgresSQL and MySQL) and to organize my projects I like the system used by Python and Django framework in order to separate each module into a folder which contains the MVC architecture. In Django, it is called MVT (Model, View, Template).

Currently, I am studying software architectures, REST API’s and GRAPHQL, for example, together with NodeJS. I chose NodeJs to improve my skills in JavaScript, because it is used in browsers, frontend frameworks (Vuejs, ReactJs), and it builds desktop and mobile applications. I understand concepts about REST API’s and GRAPHQL and I want to develop a system using them. The next goal is to learn about microservices and Queque Systems.
